# 🚀 Smart Pipeline Monitoring Dashboard

An interactive data analytics dashboard built using Streamlit to monitor pipeline pressure, detect anomalies, and visualize trends in real time.

---

## 📊 Features

- 📈 Time-series visualization of pressure data
- 🔍 Anomaly detection using statistical thresholds
- 🔥 Heatmap for pattern analysis (minute vs hour)
- 🎛️ Interactive filtering (by hour range)
- 🚨 Alert system highlighting high-pressure events
- 📊 Key metrics (Average, Max, Min, Alerts count)

---

## 🧠 How It Works

- Simulated pipeline sensor data is processed using Python
- Statistical logic (mean + standard deviation) is used for anomaly detection
- Data is visualized using Streamlit with line charts and heatmaps
- Alerts are generated when pressure exceeds threshold conditions

---

## 🛠️ Tech Stack

- Python
- Pandas & NumPy
- Streamlit
- Matplotlib & Seaborn
